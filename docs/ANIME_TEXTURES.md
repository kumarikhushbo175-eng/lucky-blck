# Anime PNG Textures Guide

## Lucky Block Anime Textures

This directory contains anime-style PNG textures for the Lucky Block mod.

### Texture Files

The following anime-style textures need to be created as 16x16 PNG images:

#### Main Block Textures
- **lucky_block_top.png** - Top face with anime-style lucky symbol
- **lucky_block_side.png** - Side face with anime characters
- **lucky_block_bottom.png** - Bottom face with decorative anime art

#### Particle Effects (Optional)
- **sparkle.png** - Anime sparkle effect
- **aura.png** - Anime aura effect
- **star.png** - Anime star particle

### How to Add Anime Textures

1. **Create 16x16 PNG images** with anime art style
2. **Place in:** `resource_pack/textures/blocks/`
3. **Format:** PNG with transparency support
4. **Color Palette:** Bright, vibrant anime colors

### Recommended Anime Styles

- **Lucky Symbol:** Gold/yellow star with anime eyes
- **Side Faces:** Cute anime characters (cat, girl, boy)
- **Effects:** Glowing auras, sparkles, magical symbols

### Creating Custom Anime Textures

You can use tools like:
- Aseprite (paid, specialized for pixel art)
- Piskel (free, online)
- Krita (free, powerful)
- Photoshop (paid)

### Texture Specifications

```
Format: PNG (RGBA)
Size: 16x16 pixels
Color Depth: 32-bit
Transparency: Yes (alpha channel)
```

### File Locations

```
resource_pack/
└── textures/
    └── blocks/
        ├── lucky_block_top.png
        ├── lucky_block_side.png
        ├── lucky_block_bottom.png
        ├── sparkle.png

```

### Tips for Anime Style

1. Use bright, vibrant colors
2. Add outlines for character definition
3. Include expressive anime eyes
4. Use gradients for depth
5. Add small details for charm

### Testing Textures

1. Place PNG files in the correct folder
2. Load the resource pack in Minecraft
3. Place/break a Lucky Block
4. Verify textures display correctly
