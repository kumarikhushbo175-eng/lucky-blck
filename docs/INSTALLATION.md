# Installation Guide

## Prerequisites
- Minecraft Bedrock Edition 1.20 or higher
- A device running Windows 10/11, Nintendo Switch, Xbox, or mobile

## Installation Steps

### Option 1: Using .mcaddon File (Easiest)

1. Download the `LuckyBlock.mcaddon` file
2. Double-click the file (Windows) or tap it (mobile)
3. Minecraft will automatically import both packs
4. The packs will appear in your world settings

### Option 2: Manual Installation

1. **Download the packs** from the releases page
2. **Extract the files** to:
   - **Windows**: `%appdata%/Microsoft/Windows/AppsLocal/Microsoft.MinecraftUWP_8wekyb3d8bbwe/Local State/games/com.mojang/behavior_packs` and `resource_packs`
   - **Mobile**: Use a file manager to locate Minecraft's data folder
   - **Nintendo Switch/Xbox**: Use the in-game pack manager

3. **Open Minecraft** and create a new world
4. **Go to World Settings** → **Behavior Packs** → Add "Lucky Block Behavior Pack"
5. **Go to Resource Packs** → Add "Lucky Block Resource Pack"
6. **Confirm changes** and start playing!

## Enabling Experimental Features

Some features may require experimental features enabled:

1. Create a new world
2. Go to Settings → Experimental
3. Enable all experimental toggles
4. Apply and start playing

## Troubleshooting

### The pack doesn't appear
- Make sure you're using Minecraft 1.20 or higher
- Try restarting Minecraft
- Check that files are in the correct directory

### Custom textures not showing
- Ensure the Resource Pack is loaded
- Check that the Resource Pack is loaded BEFORE the Behavior Pack
- Restart Minecraft

### Commands not working
- Enable Cheats in world settings
- Make sure you have operator permissions

## Getting Help

If you encounter issues:
1. Check the troubleshooting section above
2. Create an issue on GitHub
3. Contact the mod developer

## Uninstallation

1. Go to World Settings
2. Remove both "Lucky Block" packs
3. Restart Minecraft
4. Delete the pack folders from your system if desired
