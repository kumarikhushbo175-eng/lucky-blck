# Lucky Block Mod - Changelog

## Version 1.0.0 (Initial Release)

### Features Added
- ✨ Lucky Block - A special golden block with random rewards
- 💎 Reward System with configurable loot table
- 🎨 Custom textures and particle effects
- 🔊 Sound effects for block breaking
- 📦 Behavior pack for game mechanics
- 🎭 Resource pack for visuals

### Rewards Included
- Diamonds (3-8)
- Emeralds (5-15)
- Netherite Ingots (1-4)
- Enchanted Golden Apples
- Enchanted Diamond Sword (15-30)
- Enchanted Diamond Pickaxe (15-30)
- Mob spawns (Iron Golems, Wolves, Horses)
- Potion effects (Strength, Speed)
- TNT and explosions

### Supported Versions
- Minecraft Bedrock Edition 1.20+
- Windows 10/11
- iOS/Android
- Nintendo Switch
- Xbox One/Series X|S

### Known Limitations
- Requires experimental features for some effects
- Custom sounds need to be added separately
- Textures need to be created in 16x16 PNG format
