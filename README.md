# Lucky Block Mod for Minecraft Bedrock

A feature-rich Lucky Block mod that brings exciting random events and rewards to Minecraft Bedrock Edition.

## Features

- **Random Rewards**: Breaking a Lucky Block gives you random loot and effects
- **Special Effects**: Experience potion effects, explosions, mob spawns, and more
- **Customizable**: Easy to configure reward probabilities
- **Multiplayer Support**: Works in multiplayer worlds
- **Beautiful Design**: Custom textures and animations

## Installation

1. Download the latest release
2. Open the `.mcaddon` or import the resource and behavior packs
3. Enable the pack in your world settings
4. Start playing!

## How to Use

- Find or craft Lucky Blocks (Gold Block recipe)
- Break them to get random rewards
- Each break triggers a unique event

## Contents

- `behavior_pack/` - Behavior pack files
- `resource_pack/` - Resource pack files
- `docs/` - Documentation

## Rewards Include

- Diamonds and valuable ores
- Enchanted tools and weapons
- Potion effects (Speed, Strength, etc.)
- Rare mobs
- Explosions and TNT traps
- Weather effects
- Teleportation
- And much more!
